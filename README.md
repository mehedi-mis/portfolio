# Professional Portfolio — Md. Shakhaout Hossain Rimon

A high-performance, aesthetically premium personal portfolio website built with modern web technologies. This project showcases my professional journey as a **Software Engineer (R&D)**, highlighting expertise in backend architecture, microservices, and competitive programming.

---

## 🚀 Key Features

- **Modern UI/UX**: Implements a sleek dark-themed design with glassmorphism effects and high-fidelity typography.
- **Dynamic Background**: Interactive canvas-based particle network that responds to user presence.
- **Smooth Animations**: Powered by Framer Motion for staggered entry, scroll-based triggers, and layout transitions.
- **Real-time Contact Integration**: Integrated with **FormSubmit API** for instant email delivery directly from the contact form.
- **Filterable Showcase**: Project gallery featuring category filters (Spring Boot, Python, ML) and expandable technical details.
- **Professional Timeline**: Experience and education sections displayed via a clean timeline architecture.
- **Fully Responsive**: Optimized for every device, from mobile handsets to ultra-wide displays.

## 🛠️ Tech Stack

- **Frontend Core**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Vanilla CSS (CSS3 variables, Flexbox/Grid)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Email Service**: [FormSubmit](https://formsubmit.co/)

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-site
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### Production Build

Create an optimized build for deployment:
```bash
npm run build
```
The output will be in the `dist/` directory, ready to be hosted on Vercel, Netlify, or GitHub Pages.
