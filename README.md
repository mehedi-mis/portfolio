## Getting Started

### Prerequisites

- Node.js 22 recommended
- npm
- `nvm` recommended for managing Node versions

### Upgrade Node.js

Vite in this project reports that Node.js `20.19+` or `22.12+` is required. Easiest with `nvm`:

```bash
nvm install 22
nvm use 22
nvm alias default 22
node -v
```

### Install Dependencies

After upgrading Node, reinstall dependencies:

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The local dev server will be available at `http://localhost:5173`.

### Production Build

```bash
npm run build
```

Build output is generated in `dist/`.
